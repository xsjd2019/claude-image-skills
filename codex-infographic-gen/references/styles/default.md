# default — 明るいプレゼン資料調

このプリセットは、白背景にパステル色のカードと鮮やかなアクセントカラーを組み合わせ、フラットなベクターイラストを添えた、明るく親しみやすいトンマナ。プレゼン資料、SNS投稿、ブログのアイキャッチなどにそのまま使える質感。

## 配色

### ベース
- 背景: 白 #FFFFFF
- 区切り線・薄い罫線: ライトグレー #E5E5E5

### カード背景（複数カード並列時はパステル4色をローテーション）
- ペールブルー   #E3EFFB
- ペールグリーン #E6F7E9
- ペールパープル #F0E5F7
- ペールベージュ #F5F2EE

### アイコン・アクセント（鮮やかな4色）
- ビビッドブルー  #3F7BE8
- ビビッドグリーン #3BBE5C
- ビビッドパープル #A150D2
- ダークグレー   #4A4A4A

### 強調・装飾
- タイトル強調語・赤い箇条書きドット・下線アクセント: 赤 #E63B3F
- サブタイトル枠（ピル型）: 赤系の細い枠線、または赤背景に白文字
- CTAボックス（最下部）: 紫から青緑のグラデーション #5B3CDC → #4DC5BA、文字は白
- コーナー装飾の小円: ペールブルーやペールパープルの淡色

## タイポグラフィ

- フォント: Noto Sans JP / 游ゴシック相当のサンセリフ和文。**明朝NG**
- タイトル: 極太（Bold/Black）、大きく、紙面の最上部で主役。重要キーワードだけ赤で強調可
- サブタイトル: 中太、ピル型の赤い枠で囲うか、赤背景の角丸ピルに白文字
- 見出し（カード内）: 太め、アイコン横に配置
- 本文・箇条書き: ミディアム、3行以内推奨。箇条書きは赤いドット
- CTAコピー: 極太、白文字、グラデーション背景の上

## レイアウト

- 全体は白背景、要素はカード単位で角丸枠（border-radius 16〜24px相当）
- カード並列のときは 2×2 グリッドや 2列が自然
- 各カードは「アイコン + 見出し + 箇条書き + シーンイラスト」を基本構成として変化を持たせる（ただし**全カードを同じ形にしない**こと）
- セクション間の余白は適度に取る（要素同士が窮屈にならない程度）。ただし **紙面の左右・上下のマージンは狭く**（30〜50px 程度）、紙面の有効エリアはコンテンツでしっかり埋める
- 紙面下部に CTA や結論ボックスを置く構成が映える（グラデーション枠）

## イラスト

- フラットベクタースタイル
- 親しみやすい人物中心（学習中・仕事中・考え中・対話中などの動作付き）
- カラフルだが派手すぎない。アクセントカラーで彩度を持たせる
- 各カードに小さなシーンイラスト（人物 + 関連オブジェクト）を添える
- 顔の表情は控えめ（記号的）でOK
- 業務系の小道具（PC、書類、グラフ、地球儀、植木鉢、ブラウザウィンドウ等）を組み合わせる

## 装飾

- コーナー（左上・右上など）にパステル色の小円や輪っかを散らす（控えめに）
- タイトル下に赤系の下線アクセントや短い装飾線
- 重要箇所はピル型枠・角丸枠で囲んで強調
- グラデーションは紙面 1〜2 箇所に限定（メインCTAや見出し帯など）

## 全体の印象

- **明るい・親しみやすい・カジュアル寄り**
- 情報密度は保ちつつ、ビジュアルが豊か（テキストだけにならない）
- プレゼンスライドにそのまま貼れる質感
- SNS投稿用のアイキャッチとして使っても映える
- 読み手が「読みたくなる」エンタメ性を備える
